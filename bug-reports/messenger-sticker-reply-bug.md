ID: BUG-002
Título: Responder a sticker genera error visual en el chat de Messenger

Severidad: Media
Prioridad: Baja
Entorno: Messenger v450.0 en Android

Descripción:
Al responder un sticker en un chat, el formato del mensaje de respuesta se muestra incorrecto.

Pasos para Reproducir:
1. Abrir cualquier conversación en Messenger
2. Enviar un sticker 
3. Mantener presionado el sticker y seleccionar "Responder"
4. Escribir texto de respuesta y enviar

Resultado Actual:
El texto de respuesta aparece superpuesto o con tamaño incorrecto junto al sticker

Resultado Esperado:
El mensaje de respuesta debería mostrarse debajo del sticker citado, con formato normal

Evidencia: Pantallazo pendiente
